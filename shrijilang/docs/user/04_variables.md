# Variables

Version: 1.0

---

## What Is A Variable?

Variable ek naam hai jiske andar hum value store kar sakte hain.

ShrijiLang mein variable banane ke liye `mavi` keyword use hota hai.

---

## Creating A Variable

Example:

```sri
mavi age = 20

bolo age
```

Output:

```text
20
```

---

## Storing Text

```sri
mavi naam = "Radhe"

bolo naam
```

Output:

```text
Radhe
```

---

## Storing Numbers

```sri
mavi x = 10
mavi y = 5

bolo x
bolo y
```

---

## Using Variables In Expressions

```sri
mavi a = 10
mavi b = 5

mavi total = a + b

bolo total
```

Output:

```text
15
```

---

## Updating Variables

```sri
mavi count = 1

count = count + 1

bolo count
```

Output:

```text
2
```

---

## What Did We Learn?

* `mavi` variable banata hai
* Variables text aur numbers store kar sakte hain
* Variables expressions mein use ho sakte hain
* Variables update kiye ja sakte hain
