---

## Your First Program

Programming ki duniya mein pehla program traditionally "Hello World" hota hai.

ShrijiLang mein output dikhane ke liye `bolo` keyword use kiya jata hai.

Example:

```sri
bolo "Namaste"
```

Output:

```text
Namaste
```

---

## Printing Values

Sirf text hi nahi, variables bhi print kiye ja sakte hain.

```sri
mavi naam = "Radhe"

bolo naam
```

Output:

```text
Radhe
```

---

## Multiple Outputs

```sri
bolo "Namaste"

bolo "ShrijiLang"
```

Output:

```text
Namaste
ShrijiLang
```

---

## What Did We Learn?

Is chapter mein humne seekha:

* `bolo` output dikhata hai
* Text print kiya ja sakta hai
* Variables print kiye ja sakte hain
