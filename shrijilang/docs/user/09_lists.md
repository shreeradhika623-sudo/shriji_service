# Lists

Version: 1.0

---

## What Is A List?

List multiple values ko ek jagah store karne ka tareeka hai.

Aap numbers, text aur dusri values ko ek list ke andar rakh sakte hain.

---

## Creating A List

```sri
mavi nums = [1, 2, 3]
```

---

## Accessing Values

List ke elements index se access kiye jaate hain.

Index 0 se start hota hai.

Example:

```sri
mavi nums = [1, 2, 3]

bolo nums[0]
bolo nums[1]
bolo nums[2]
```

Output:

```text
1
2
3
```

---

## Index Positions

```text
Index:   0   1   2
Value:   1   2   3
```

---

## Why Use Lists?

Without a list:

```sri
mavi a = 1
mavi b = 2
mavi c = 3
```

With a list:

```sri
mavi nums = [1, 2, 3]
```

---

## What Did We Learn?

* Lists multiple values store kar sakti hain
* Index 0 se start hota hai
* Square brackets `[]` use hote hain
