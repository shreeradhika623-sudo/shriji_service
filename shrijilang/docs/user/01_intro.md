# Introduction to ShrijiLang 🌸

Version: 1.0
Audience: Beginners
Prerequisite: None

---
## Welcome

ShrijiLang ek human-friendly programming language hai jo beginners ko programming step-by-step samjhane ke liye banayi gayi hai.

Iska objective traditional programming languages ko replace karna nahi hai.

Iska objective programming ko samajhna, explain karna aur human language ke kareeb lana hai.

---

## Why ShrijiLang?

Programming seekhte waqt beginners ko do problems aati hain:

1. Concepts samajhna
2. Compiler errors samajhna

Aksar doosri problem pehli problem se bhi badi ban jaati hai.

ShrijiLang is experience ko simple banane ki koshish karti hai.

Language ka focus sirf code chalana nahi hai.

Language ka focus code samjhana bhi hai.

---

## Philosophy

ShrijiLang ek simple philosophy follow karti hai:

* Ek problem
* Ek permanent solution
* Full testing
* Fir next problem

Isi approach se language ka development kiya gaya hai.

---

## First Example

```sri
bolo "Namaste"
```

Output:

```text
Namaste
```

---

## Variables

Variables values store karne ke liye use hote hain.

```sri
mavi age = 20
mavi name = "Shri"

bolo age
bolo name
```

---

## Conditions

Decision lene ke liye conditions use ki jaati hain.

```sri
mavi age = 20

agar (age >= 18) {

    bolo "Adult"

} warna {

    bolo "Minor"

}
```

---

## Loops

Same kaam ko repeat karne ke liye loops use hote hain.

```sri
mavi x = 10

jabtak (x >= 1) {

    bolo x

    x = x - 1

}
```

---

## Functions

Reusable logic ko functions ke andar rakha ja sakta hai.

```sri
rachna add(a, b)
{
    bolo a + b
}

add(5, 8)
```

---

## Real-Life Example

ShrijiLang simple calculators aur learning projects banane ke liye bhi use ki ja sakti hai.

```sri
mavi a = 10
mavi b = 5

mavi add = a + b
mavi sub = a - b

bolo add
bolo sub
```

---

## Error Philosophy

ShrijiLang beginner-friendly errors par bahut focus karti hai.

Language ka goal sirf problem dikhana nahi hai.

Language ka goal user ko samjhana hai:

* Kya hua
* Kyun hua
* Kaise theek karna hai

ShrijiLang user ke against nahi hai.

ShrijiLang aur user milkar problem solve karte hain.
