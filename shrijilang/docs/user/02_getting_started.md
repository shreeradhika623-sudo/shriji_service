# Getting Started

Version: 1.0

---

## Welcome

Aapne ShrijiLang ke baare mein jaan liya.

Ab hum apna pehla program chalane ke liye taiyar hain.

Is guide mein aap seekhenge:

* ShrijiLang kaise start karein
* Apna pehla program kaise likhein
* Output kaise dekhein
* Documentation aur help kaise use karein

---

## Running ShrijiLang

Agar ShrijiLang install hai to terminal mein:

```bash
shrijilang
```

ya

```bash
./sri
```

run kijiye.

Aapko Shriji shell dikhegi.

Example:

```text
🌸 Shriji Shell 🌸

Type 'exit' to quit

Shiri>
```

---

## Your First Command

Try:

```sri
bolo "Radhe Radhe"
```

Output:

```text
Radhe Radhe
```

---

## Using Help

Future versions mein documentation directly shell ke andar bhi available hogi.

Examples:

```sri
madad
```

```sri
madad "variables"
```

```sri
help "loops"
```
