# Dictionaries

Version: 1.0

---

## What Is A Dictionary?

Dictionary key-value pairs store karta hai.

Har value ek unique key ke saath store hoti hai.

---

## Creating A Dictionary

```sri
mavi user = {
    "naam": "Radhe",
    "umar": 25
}
```

---

## Accessing Values

Dictionary values keys ke through access ki jaati hain.

```sri
mavi user = {
    "naam": "Radhe",
    "umar": 25
}

bolo user["naam"]
bolo user["umar"]
```

Output:

```text
Radhe
25
```

---

## Dictionary Structure

```text
"naam"  → "Radhe"
"umar"  → 25
```

---

## Why Use Dictionaries?

List position yaad rakhti hai.

Dictionary meaning yaad rakhti hai.

Example:

```sri
mavi user = {
    "naam": "Radhe",
    "umar": 25,
    "shahar": "Vrindavan"
}
```

---

## What Did We Learn?

* Dictionaries key-value pairs store karti hain
* Keys strings ho sakti hain
* Values keys ke through access ki jaati hain
