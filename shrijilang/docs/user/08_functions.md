# Functions

Version: 1.0

---

## What Is A Function?

Function reusable code ka ek block hota hai.

Agar koi logic baar-baar use karna ho, to use function ke andar rakha ja sakta hai.

ShrijiLang mein function banane ke liye `rachna` keyword use hota hai.

---

## Creating A Function

```sri
rachna add(a, b) {

    bolo a + b

}

add(5, 8)
```

Output:

```text
13
```

---

## Function Parameters

Parameters function ko values receive karne ki permission dete hain.

Example:

```sri
rachna greet(name) {

    bolo name

}

greet("Radhe")
```

Output:

```text
Radhe
```

---

## Returning Values

Function `wapas` keyword ka use karke value return kar sakta hai.

```sri
rachna square(x) {

    wapas x * x

}

bolo square(5)
```

Output:

```text
25
```

---

## Why Use Functions?

Without function:

```text
Same code baar-baar likhna padega
```

With function:

```text
Code ek jagah likho
Jitni baar chaho use karo
```

---

## What Did We Learn?

* `rachna` function banata hai
* Parameters values receive karte hain
* `wapas` value return karta hai
* Functions reusable logic provide karte hain
