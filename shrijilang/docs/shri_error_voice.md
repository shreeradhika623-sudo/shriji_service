# SHRI ERROR VOICE V1

## Mission

Shriji sirf error nahi batayegi.

Shriji user ko samjhayegi:

1. Kya hua
2. Kyun hua
3. Kaise theek karna hai

User aur Shriji ek team hain.

Problem enemy hai.

User nahi.

--------------------------------------------------

## Personality

Shriji hai:

- Calm
- Intelligent
- Confident
- Patient
- Helpful

Shriji kabhi:

- User ko blame nahi karegi
- Gussa nahi karegi
- Judge nahi karegi
- Panic nahi karegi

--------------------------------------------------

## Standard Format

HINT: (optional)

ERROR:
Kya hua

DETAILS:
Kyun hua

EXAMPLE:
(optional)

SUGGESTION:
Ab kya karna hai

--------------------------------------------------

## Golden Rule

Wrong:

"tumne galat syntax likha"

Correct:

"is expression ko samajhne me dikkat hui"

--------------------------------------------------

## Banned Words

User-facing messages me avoid:

- token
- identifier
- parser
- AST
- operand
- lexical

--------------------------------------------------

## Examples

Bad:

ERROR:
bad token

Good:

ERROR:
is jagah mujhe kuch unexpected mila

DETAILS:
main is expression ko samajh nahi paayi

SUGGESTION:
expression ko dobara check kijiye

--------------------------------------------------

Bad:

ERROR:
1 argument required

Good:

ERROR:
abs() ko ek value chahiye

DETAILS:
bracket ke andar koi value nahi mili

SUGGESTION:
andar ek number likhiye

--------------------------------------------------

Final Philosophy

Har language code chalati hai.

Shriji samjhati hai.
